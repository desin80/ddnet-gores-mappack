# DDNet Gores 地图包

语言切换：[中文](README.md) | [English](README.en.md)

该仓库包含了个人收集的 gores 地图。

> 说明：metadata 可能不是 100% 准确。

## 内容

- `maps/`
  原始 `.map` 地图文件。
- `maplists/`
  按难度分类的 JSON 列表文件。
- `metadata/all-metadata.json`
  汇总后的全部地图元信息。
- `metadata/stats.json`
  各难度地图数量统计。
